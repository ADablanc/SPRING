Chromium Portable 9.0.597.19 Help

Chromium Portable is the Chromium web browser packaged as a portable app, so you can run it from an iPod, USB flash drive, portable hard drive, CD or any other portable media. You can take your bookmarks, extensions and saved passwords with you, and it will leave no personal information behind on the machine you run it on.

For more information, please check the Chromium Portable homepage. Check the licenses file for redistribuition conditions.
